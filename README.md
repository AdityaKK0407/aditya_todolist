## Description

This is a simple web-based To-Do list application built with Go. It allows users to manage their tasks through a web interface, leveraging a RESTful API for task operations and serving dynamic content via HTML templates and static assets.

## Features

*   **Task Management:** Create, read, update, and delete (CRUD) To-Do items.
*   **Web Interface:** User-friendly web interface for interacting with the To-Do list.
*   **RESTful API:** Backend API for programmatic access to To-Do list functionalities.
*   **Static File Serving:** Serves CSS and JavaScript files for the frontend.
*   **HTML Templating:** Dynamic rendering of web pages using Go's `html/template` package.

## Technologies Used

*   **Backend:** Go (Golang)
*   **Frontend:** HTML, CSS, JavaScript
